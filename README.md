# Todo List App (React + TypeScript)

This is a **modern Todo List application** built using **React** and **TypeScript**.  
It allows users to **add**, **edit**, and **delete** tasks in a visually appealing interface. The app uses **React hooks** for state management and is styled creatively using CSS.

---

## Features

- Add new tasks
- Edit existing tasks
- Delete tasks
- Interactive and modern design
- Responsive and user-friendly interface

---

## Technologies Used

- React
- TypeScript
- Vite (for project setup)
- CSS (modern styling with hover effects and transitions)
